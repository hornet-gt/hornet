Detailed Description

template<typename T, unsigned BLOCK_ITEMS, unsigned BLOCKARRAY_ITEMS>
class custinger::BitTree< T, BLOCK_ITEMS, BLOCKARRAY_ITEMS >

Vectorized Bit Tree

Internal representation
[1011] // root –> internal
[0001] [0000] [1001] [0110] // –> internal
[....] [....] [....] [....] [....] ... // last_level –> external

Remarks
1 means block available, 0 block used
Template Parameters
BLOCK_ITEMSNumber of edges per block. Maximum number of items that that can fit in a single block
BLOCKARRAY_ITEMSNumber of edges per BlockArray. Maximum number of items that that can fit in a single BlockArray
Precondition
BLOCK_ITEMS \(\le\) BLOCKARRAY_ITEMS
